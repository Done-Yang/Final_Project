                ໂຄງຮ່າງບົດໂຄງການ
I.  ຊື່ຫົວບົກ
    1. ລາຍລະອຽດບົດໂຄງການ
        ຊື່ຫົວÚດ : ການສ້າງເວັບໄຊຄົ້ນຫາລາຍລະອຽດຂອງ ທີ່ຢູ່ IP ດ້ວຍ 
    2. ສະມາຊິກ
        ນາງ ປຸກກີ້ ສຸຂະຜັນດິດ
        ທ້າວ ດອນ ຢ່າງ
        ນາງ ຕ້ອຍມະນີ
    3. ສາຂາ : ວິທະຍາສາດຄອມພິວເຕີ(IT)
    4. ອາຈານທີ່ປຶກສາ : ອາຈານ ສົມຈິດ ຂັນທະວົງ
    5. ສົກຮຽນ 2023-2024
II. ຄວາມສຳຄັນ ແລະ ທີ່ມາຂອງການສຶກສາ
    (ຫຼັກການ ຫຼື ເຫດຜົນທີ່ເຮົາຄຕ້ອງການສຶກສາ)
    (1. ຄວາມສຳຄັນຂອງອີນເຕີເນັດ)
        ອິນເຕີເນັດ (Internet) ທີ່ຫມາຍຄວາມວ່າ ເຄືອຂ່າຍຄອມພິວເຕີ ນັ້ນແມ່ນຂະຫນາດໃຫຍ່ ມີການເຊື່ອມຕໍ່ລະຫວ່າງເຄືອຂ່າຍຫຼາຍທົ່ວໂລກ.
    ໂດຍການນຳໃຊ້ພາສາທີ່ໃຊ້ໃນການຕິດຕໍ່ສື່ສານລະຫວ່າງຄອມພິວເຕີທີ່ເອີ້ນວ່າ ໂປໂຕຄອນ (Protocol) ຜູ້ໃຊ້ເຄືອຂ່າຍນີ້ສາມາດຕິດຕໍ່ສື່ສານກັນໄດ້ຫຼາຍວິທີເຊັ່ນ:
    ອີເມວ, ກະດານເວັບ ແລະສາມາດຄົ້ນຫາຂໍ້ມູນ ແລະ ຂ່າວຕ່າງໆ ລວມທັງການຄັດລອກໄຟລ໌ຂໍ້ມູນ ແລະ ໂປຣ ແກຣມ ຕ່າງໆ ໄດ້. ເຊີ່ງມີຄວາມສຳຄັນໃນ
    ຫຼາຍດ້ານເຊັ່ນ: ການສຶກສາ, ທຸລະກິດ, ການຄ້າ, ຄວາມບັນເທີງ ແລະ ທຸກສີ່ງທຸກຢ່າງໃນປັດຈຸບັນນີ້ລ້ວນແລ້ວຕ້ອງການອີນເຕີເນັດ

    (2. ຄວາມສຳຄັນຂອງເວັບໄຊເຮົາ)
        ເມື່ອໂລກມີການເຊື່ອມໂຍງກັນຫຼາຍຂຶ້ນ, ຄວາມສໍາຄັນຂອງເວັບໄຊທ໌ຊອກຫາ IP ໄດ້ເພີ່ມຂຶ້ນຢ່າງຫຼວງຫຼາຍ. ເວັບໄຊທ໌ຊອກຫາ IP ແມ່ນເຄື່ອງມືທີ່ອະນຸຍາດໃຫ້ຜູ້ໃຊ້ໃນການລະບຸສະຖານທີ່
    ແລະ ຂໍ້ມູນອື່ນໆທີ່ກ່ຽວຂ້ອງກັບທີ່ຢູ່ IP ທີ່ກໍານົດໄວ້. ຂໍ້ມູນນີ້ສາມາດນໍາໃຊ້ເພື່ອຈຸດປະສົງທີ່ຫຼາກຫຼາຍ, ລວມທັງການສືບສວນຄວາມປອດໄພ, ການຄົ້ນຄວ້າການຕະຫຼາດ, ແລະການວິເຄາະເວັບໄຊທ໌.
    ຫນຶ່ງໃນການນໍາໃຊ້ທີ່ສໍາຄັນທີ່ສຸດຂອງເວັບໄຊທ໌ຊອກຫາ IP ແມ່ນສໍາລັບການສືບສວນຄວາມປອດໄພ. ໂດຍການກໍານົດສະຖານທີ່ແລະຂໍ້ມູນອື່ນໆທີ່ກ່ຽວຂ້ອງກັບທີ່ຢູ່ IP, 
    ຜູ້ສືບສວນສາມາດຕິດຕາມໄພຂົ່ມຂູ່ທີ່ອາດຈະເກີດຂື້ນແລະດໍາເນີນການເພື່ອປົກປ້ອງລະບົບຂອງພວກເຂົາ. ນີ້ແມ່ນສິ່ງສໍາຄັນໂດຍສະເພາະສໍາລັບທຸລະກິດແລະອົງການຈັດຕັ້ງທີ່ອີງໃສ່ອິນເຕີເນັດສໍາລັບການປະຕິບັດງານຂອງພວກເຂົາ,
    ຍ້ອນວ່າການໂຈມຕີທາງອິນເຕີເນັດແລະໄພຂົ່ມຂູ່ອື່ນໆສາມາດເຮັດໃຫ້ເກີດຄວາມເສຍຫາຍຢ່າງຫຼວງຫຼາຍຕໍ່ການດໍາເນີນງານຂອງພວກເຂົາ.
